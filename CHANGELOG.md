# Changelog

## [0.1.0] - 2026-05-26

### Added
- Initial release
- SRT file parsing and serialization
- LLM translation via Ollama and LM Studio providers
- Google Translate fallback
- 8 target languages (zh-TW, ja, ko, es, fr, de, pt, ru)
- Context-aware translation with full subtitle script context
- Multi-language tabs with inline editing
- Row operations: move up/down, split, merge with above
- Multi-format export (SRT, VTT, ASS, JSON, ZIP)
- Docker support with docker-compose
- Drag-and-drop SRT file loading

[0.1.0]: https://github.com/dcchanInSz/qwen-srt-translate/releases/tag/v0.1.0
