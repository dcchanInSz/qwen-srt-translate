# Security Policy

If you discover a security vulnerability, please **do not open a public issue**.

## Security Considerations

This application is designed to run locally. It connects to local LLM servers
(Ollama, LM Studio) and the Google Translate API. No data is sent to external
servers beyond these configurable endpoints.
